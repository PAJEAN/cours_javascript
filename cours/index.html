Le JavaScript

JavaScript est le langage qui donne vie aux pages web. Là où HTML définit la structure et CSS le style, JavaScript permet d'ajouter de l'interactivité, de la logique et du dynamisme. Avec lui, un site peut réagir aux actions de l'utilisateur, mettre à jour son contenu sans recharger la page, afficher des animations ou même héberger de véritables applications complètes (jeux, outils collaboratifs, etc.). Mais JavaScript ne se limite plus au navigateur : grâce à des environnements comme Node.js, il permet également de créer des applications côté serveur, des outils bureautiques (Electron), ou même des applications mobiles (React Native). Ce langage polyvalent est aujourd'hui incontournable dans de nombreux domaines du développement informatique.

Dans ce cours, vous allez découvrir les bases du langage et sa syntaxe à travers des exemples pratiques. Un environnement de développement léger, comme Visual Studio Code, et un navigateur moderne (Chrome, Firefox, Opera…) suffisent pour expérimenter.

Les extraits de code peuvent être exécutés directement grâce aux cellules interactives intégrées dans le cours, de la même manière qu'un notebook. Certains exemples ne sont pas fonctionnels : c'est volontaire, car ils servent à illustrer des erreurs ou de mauvaises pratiques à éviter.
